# 2.介绍 #

- 项目通过React Native，仿照网易音乐播放器，简单的写了一些界面和音乐播放功能                                                
- 界面由React Native实现，播放功能用原生代码编写，通过之间交互完成播放
- 目前仅简单实现播放本地和网络音乐，后续还需要增加功能和优化
